# Matlab

## 矩阵和数组

### 1.行向量

a = 1×4

```matlab
a = [1 2 3 4]
```

### 2.列向量

```
a = [1 2 3; 4 5 6; 7 8 10]
```
a = 3×3

    1     2     3
     4     5     6
     7     8    10

#### （1）加减法
```
a + 10
```
ans = 3×3

    11    12    13
    14    15    16
    17    18    20
#### （2）三角函数
```
sin(a)
```
```
ans = 3×3

    0.8415    0.9093    0.1411
   -0.7568   -0.9589   -0.2794
    0.6570    0.9894   -0.5440
```

#### （3）转置

```
a'
```

```
ans = 3×3

     1     4     7
     2     5     8
     3     6    10
```

#### （4）逆——>单位矩阵

```
p = a*inv(a)
```

```
p = 3×3

    1.0000         0   -0.0000
         0    1.0000         0
         0         0    1.0000
```

#### （5）级乘 .

元素级乘法

```
p = a.*a
```

```
p = 3×3

     1     4     9
    16    25    36
    49    64   100
```

各个元素的三次方

```
a.^3
```

```
ans = 3×3

           1           8          27
          64         125         216
         343         512        1000
```

### 4.串联

#### 水平串联

```
A = [a,a]
```

```
A = 3×6

     1     2     3     1     2     3
     4     5     6     4     5     6
     7     8    10     7     8    10
```

#### 垂直串联

```
A = [a; a]
```

```
A = 6×3

     1     2     3
     4     5     6
     7     8    10
     1     2     3
     4     5     6
     7     8    10
```
